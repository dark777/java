Description

    The CoolServets.com Email package is a group of classes that makes it easy to send email from any Java application, applet or servlet.
Features
  • Simple to include in a Java app, applet or servlet.
  • Uses SMTP to send email.
  • Includes support for CC and BCC fields.
  • Supports Reply-To field.
  • Models the JavaMail API (so programmers familiar with JavaMail will be able to adapt this package for their use quickly).
Download & Installation
    These classes are available as a downloadable package which includes a README, Change Log (a list of what has changed in each version), the full source, the files (compiled source) you need to use the package, a single JAR file with all the classes in it, and demonstrations of how to use the classes.
