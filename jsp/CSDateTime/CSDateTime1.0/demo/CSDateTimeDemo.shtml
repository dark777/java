CSDateTime Demo
CoolServlets.com


This page uses server-side includes (SSI) to demonstrate uses of the CSDateTime servlet. See your server documentation for information about SSI. If everything is configured correctly and you are viewing this page through your webserver, you should see the date and time displayed below in a variety of formats. If nothing appears in the Date and Time column, then SSI or the servlet are not configured correctly.

For a full list of valid parameters for the servlet, see the Readme file.

Date and Time SSI Syntax
<SERVLET CODE=CSDateTime> </SERVLET>
<SERVLET CODE=CSDateTime>
<PARAM name="format" value="MMM - d - yyyy">
</SERVLET>
<SERVLET CODE=CSDateTime>
<PARAM name="format" value="h:mm a, zzzz">
</SERVLET>
<SERVLET CODE=CSDateTime>
<PARAM name="format" value="h:mm a, zzzz">
<PARAM name="timeZone" value="EST">
</SERVLET>

You can also call the script without using SSI. However, that will simply print out a text string representing the date. Using the servlet that way will probably not be very useful. However, it can be a good testing mechanism if you are having problems with SSI. For example:
