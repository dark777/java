- Software -

jChatBox is a 100% JAVA JSP/Servlets chat application. Here are the three requirements you need to run it :

• Servlet Engine or Application Server compliant JSP1.0/Servlets2.1 API.
• JVM 1.2.2 or higher.
• Browser (Internet Explorer 4.x or higher, Netscape 4.x or higher)

jChatBox only needs HTTP to communicate between server-side (servlet engine) and client-side (browser).

jChatBox should run under ANY platform supporting JSP1.0/Servlets 2.1.
